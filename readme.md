Este proyecto es un generador de señales digitales que permite visualizar diferentes tipos de codificación de señales, incluyendo NRZ-L, NRZ-I, Bipolar, Manchester y Manchester Diferencial.

## Características

- Genera gráficos de señales digitales basados en una secuencia de bits proporcionada por el usuario.
- Soporta los siguientes tipos de codificación de señales: NRZ-L, NRZ-I, Bipolar, Manchester y Manchester Diferencial.
- Soporta las formulas de Shannon y Nyquist
- Tiene apuntes teoricos
